Privacy Policy & Terms for Personal Social Insights Dashboard

This is a personal, non-commercial application developed for the sole purpose of analyzing the owner's own social media data.

The application runs locally on the user's machine and does not collect, store, share, or transmit any user data to any third-party servers.

All data accessed via the TikTok API is used for read-only display within the local dashboard. Data is stored only on the user's local machine via browser storage (localStorage) and can be cleared by the user at any time.
